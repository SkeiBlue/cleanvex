# V0.3 Finances minimal

Objectif: rendre visibles les mouvements financiers et permettre les premieres liaisons avec les vehicules.

## Inclus

- Comptes financiers. Fait.
- Categories revenus/depenses. Fait.
- Operations revenus/depenses. Fait.
- Resume revenus, depenses, solde. Fait.
- Operations liees a un vehicule avec `source_module`, `source_type`, `source_id`. Fait.
- Liste simple des operations. Fait.
- Module Finances desactivable avec blocage API. Fait.

## Non inclus

- Graphiques avances.
- Import CSV.
- Budgets globaux ou projet.
- Suppression avancee avec detach source.

## Tests minimum

- Creation compte. Fait.
- Creation categorie. Fait.
- Creation depense liee vehicule. Fait.
- Solde calcule. Fait.
- Module Finances desactive bloque l'API. Fait.
