# V1.2 - Contacts minimal

## Objectif

Ajouter un carnet de contacts simple pour centraliser les personnes, organisations et prestataires utiles aux autres modules.

## Inclus

- Contacts owner-scoped.
- Creation, liste, fiche detaillee et modification API.
- Type personne/organisation.
- Email, telephone, ville, organisation, tags et notes.
- Interactions datees par contact.
- Documents lies via `document_links`.
- Recherche globale enrichie.
- Rapports simples et export ZIP enrichis.
- Module activable/desactivable.

## Tests prioritaires

- Creation d'un contact.
- Ajout d'une interaction.
- Liaison document prive.
- Recherche d'un contact.
- Module desactive inaccessible.

## Non inclus

- Synchronisation Google/Apple contacts.
- Champs CRM avances.
- Liens metier fins vers vehicules, biens ou operations.
- Import CSV/vCard.
