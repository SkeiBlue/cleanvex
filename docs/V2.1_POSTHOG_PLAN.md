# V2.1 - PostHog optionnel

## Objectif

Ajouter une observabilite produit optionnelle sans donnees sensibles.

## Inclus

- SDK `posthog-js` cote frontend.
- Initialisation uniquement si `VITE_POSTHOG_KEY` est configure.
- `autocapture` desactive.
- `capture_pageview` desactive.
- Session recording desactive.
- Identification par `user.id` uniquement.
- Evenements manuels non sensibles: login, logout, inscription, verification email, creations de modules, export ZIP et activation module.

## Configuration

```env
VITE_POSTHOG_KEY="phc_xxx"
VITE_POSTHOG_HOST="https://eu.i.posthog.com"
```

## Non inclus

- Session replay.
- Capture automatique DOM.
- Donnees personnelles dans les properties.
- Feature flags UI.
