# V0.2 Vehicules minimal

Objectif: commencer a utiliser l'application avec de vrais vehicules sans developper le module atelier complet.

## Inclus

- Module `vehicles` branche au registre modules.
- Liste des vehicules.
- Creation vehicule avec type, statut, marque, modele, annee, kilometrage.
- Fiche vehicule avec resume.
- Historique kilometrage.
- Interventions simples.
- Documents lies via `document_links`.
- Photos comme documents image prives.
- Alertes simples liees au vehicule.

## Non inclus

- Stock avance.
- Donneuses avancees.
- Partage public.
- Compatibilite automatique pieces.
- Finances completes.

## Tests minimum

- Creation vehicule. Fait.
- Modification vehicule. Fait.
- Creation intervention. Fait.
- Historique kilometrage. Fait.
- Alerte vehicule simple. Fait.
- Module Vehicles desactive bloque l'API. Fait.
- Document vehicule inaccessible sans auth. Couvert par securite documents V0.1.
- Document lie au vehicule. Fait.
- Photo privee liee au vehicule. Fait via upload image prive et contexte `photo`.
