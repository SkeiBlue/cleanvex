# V1.3 - Profil, parametres et logs

## Objectif

Exploiter les tables Core deja prevues dans le cahier des charges pour rendre le socle plus administrable.

## Inclus

- Profil utilisateur consulte et modifiable.
- Sessions refresh token visibles.
- Parametres utilisateur cle/valeur.
- Logs d'activite recents.
- Logs d'audit securite recents.
- Logs erreurs serveur recents.
- Filtre global NestJS pour journaliser les erreurs 500.
- Logs sur login, refresh, logout, logout-all, profil, parametres et activation modules.
- Export ZIP enrichi avec parametres, activite et audit.
- UI Core dans le dashboard.

## Tests prioritaires

- Lecture profil.
- Modification profil.
- Enregistrement parametre.
- Lecture activite.
- Lecture audit.
- Lecture erreurs.

## Non inclus

- Gestion multi-utilisateurs.
- Permissions avancees.
- Alerting securite automatique.
- Export filtre des logs.
