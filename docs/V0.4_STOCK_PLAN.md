# V0.4 Stock minimal

Objectif: gerer les pieces, consommables et outils de base, avec flux vers finances et vehicules.

## Inclus

- Articles stock. Fait.
- Quantites et unites. Fait.
- Localisation. Fait.
- Mouvements stock. Fait.
- Achat stock qui augmente la quantite. Fait.
- Achat stock qui cree une depense financiere si un compte est fourni. Fait.
- Sortie stock qui diminue la quantite. Fait.
- Sortie stock vers vehicule avec intervention/cout vehicule. Fait.
- Module Stock desactivable avec blocage API. Fait.

## Non inclus

- Compatibilites automatiques pieces.
- Donneuses avancees.
- Seuils avances et alertes automatiques.
- Prets outils.

## Tests minimum

- Creation article. Fait.
- Achat cree mouvement stock. Fait.
- Achat cree mouvement financier. Fait.
- Sortie diminue stock. Fait.
- Sortie affecte cout vehicule. Fait.
- Module Stock desactive bloque l'API. Fait.
