# V1.1 - Immobilier minimal

## Objectif

Ajouter un premier module immobilier exploitable sans alourdir la V1 stable.

## Inclus

- Biens immobiliers owner-scoped.
- Creation, liste et fiche detaillee.
- Statut, adresse, ville, surface, pieces et valeur estimee.
- Evenements/echeances simples par bien.
- Notification automatique pour une echeance immobiliere planifiee.
- Documents lies via `document_links`.
- Operations financieres pouvant pointer vers un bien avec `source_module = real-estate`.
- Recherche globale, rapports simples et export ZIP enrichis.

## Tests prioritaires

- Creation d'un bien.
- Creation d'une echeance.
- Liaison document prive.
- Operation financiere liee a un bien.
- Module desactive inaccessible.

## Non inclus

- Gestion locataires.
- Loyers recurrents.
- Amortissements, fiscalite ou calculs avances.
- Import de diagnostics.
